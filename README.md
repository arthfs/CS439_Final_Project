## Results
- Baseline (always home win): 46.0%
- K-Means + XGBoost: 55.2%
- Improvement: +9.2%

## Run Instructions
1. Install dependencies: `pip install pandas numpy scikit-learn xgboost matplotlib seaborn`
2. Download `results.csv` from Kaggle https://www.kaggle.com/datasets/martj42/international-football-results-from-1872-to-2017
3. Open `final_project.ipynb` in Jupyter
4. Run all cells (Kernel → Restart & Run All)

## Files
- `final_project.ipynb` - Complete analysis notebook
- `results.csv` - Dataset (download separately)
